# Tailscale

Make sure to run `docker exec -it tailscale tailscale up` to connect to the Tailscale network. 

To run on Linux

```bash
curl -fsSL https://tailscale.com/install.sh | sh
```

To download on iOS

```bash
https://apps.apple.com/us/app/tailscale/id1470499037?ls=1
```
